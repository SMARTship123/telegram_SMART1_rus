⚙️ ВРК выходит из строя по узлам — и каждый узел говорит на своём языке вибрации.

Винто-рулевая колонка под нагрузкой 24/7 диагностируется не «целиком», а по элементам:
✓ редуктор и коническая передача — по боковым полосам вокруг частоты зацепления;
✓ движительный электродвигатель — по электромагнитной сигнатуре;
✓ гребной винт и ступица — по лопастной частоте;
✓ поворотный механизм и подшипники гребного вала — по трендам.

Дефект конической передачи проявляется ростом боковых составляющих задолго до стука в колонке.

Прочитать узел вовремя — значит планировать ремонт, а не ловить колонку в доке.
